10.1.1 Retired

10.1.2 TCP/IP Network Communication Support

An implementation claiming conformance to DICOM TCP/IP Network Communication Supportshall:

  1. Meet the DICOM Upper Layers Protocol requirements as defined in Section 9.

  2. Use registered Application Context Names, Abstract Syntax Names and Transfer Syntax Names as defined for OSI Object Identifiers (ISO 8824 and ISO 9834-1).

    Note

    Annex F defines the DICOM Upper Layer Protocol encoding for the Application Context Names, Abstract Syntax Names, and Transfer Syntax Names. ISO 8825 defined encoding is not used.

  3. Use one of the published and approved RFCs defining the operation of TCP/IP over specific physical networks.
