10.4 Transfer Syntax For DICOM RLE Compression

DICOM defines the RLE Compression (see Annex G). This implies that:

  1. If an Application Entity issues an A-ASSOCIATE request where any offered Abstract Syntaxes is associated in one or more Presentation Contexts(s) with RLE compression Transfer Syntax, at least one of the Presentation Contexts that include this Abstract Syntax, shall include the DICOM Default Transfer Syntax (uncompressed).
