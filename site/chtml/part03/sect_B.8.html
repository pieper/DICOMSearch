B.8 Basic Film Box Information Object Definition

B.8.1 IOD Description

The Basic Film Box Information Object Definition is an abstraction of the presentation of one film of the film session. The Basic Film Box IOD describes the presentation parameters that are common for all images on a given sheet of film.
