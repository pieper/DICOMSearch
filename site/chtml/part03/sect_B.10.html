B.10 Basic Annotation Box Information Object Definition

B.10.1 IOD Description

The Basic Annotation Box Information Object Definition is an abstraction of the presentation of an annotation (e.g., text string) on a film. The Basic Annotation Box IOD describes the most used text related presentation parameters.
