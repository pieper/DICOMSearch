5.5 Types and Conditions in Normalized IODs

When a Normalized Information Object Definition in PS3.3 invokes Modules (e.g., the SOP Common Module) or Attribute Macros that are specified with Data Element Types, those specified Data Element Types and Conditions do not apply. Rather, the Data Element Types and Conditions have to be specified for each Attribute for both SCU and SCP in the appropriate Service definition in PS3.4.
