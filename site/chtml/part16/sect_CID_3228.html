CID 3228 ECG Timing Measurements

This Context Group include both global and per lead ECG measurements specified in the ISO/IEEE 11073-10102 MDC nomenclature. Note that the MDC codes for "per lead" measurements are base codes for post-coordination with separately encoded lead identifiers. MDC also defines pre-coordinated codes that include both the measurement and the lead, which may be used in the same context as this Context Group; see the ISO/IEEE Standard.
