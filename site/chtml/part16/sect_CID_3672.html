CID 3672 Pacemakers

This Context Group includes the full set of codes for types of pacemakers specified in the NASPE/BPEG Generic Pacemaker Code (NBG). The Coding Scheme Designator (0008,0102) shall be NBG.

Note

  1. A prior version of this context group used codes from the SCP-ECG vocabulary.

  2. Further information at http://www.hrsonline.org/News/ep-history/topics-in-depth/modecodehistory.cfm. For reference, the scheme is reproduced here:

    Code Position

    1 - Chamber(s) paced

    2 - Chamber(s) sensed

    3 - Response to sensing

    4 - Rate modulation

    5- Multisite pacing

    Code values

    O = None

    A = Atrium

    V = Ventricle

    D = Dual (A+V)

    S = Single(A or V - Mfr designation only)

    O = None

    I = Inhibited

    R = Rate modulation
