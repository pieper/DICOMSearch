CID 8130 Staining Protocols

This Context Group is intended for use in the Scheduled Protocol Code Sequence (0040,0008) and the Performed Protocol Code Sequence (0040,0260) attributes for an automated slide stainer. When so used, an Item with value (P3-00003, SRT, "Staining") will also include a Protocol Context Sequence(0040,0440) using TID 8003 “Specimen Staining” Specimen Staining to identify the specific stain substance.
