CID 7250 Multi-Frame Subset Type

This Context Group specifies the terms used to identify a subset of frames of a multi-frame image. It is used for encoding an equivalent of the Image SOP Instance Reference Macro (see PS3.3) in an HL7 v3 data structure (see HL7 v3 CMET, COCT_RM830120 "A_DicomCompositeObjectReference minimal").
