AA.2 Behavior of a SCP

For a device that is both a SCU and a SCP of these Storage SOP Classes, in addition to the behavior for the Storage Service Class specified in Section B.2.2, the following additional requirements are specified for Structured Reporting Storage SOP Classes:

  • A SCP of these SOP Class shall support Level 2 Conformance as defined in Section B.4.1.

Note

This requirement means that all Type 1, Type 2, and Type 3 Attributes defined in the Information Object Definition and Private Attributes associated with the SOP Class will be stored and may be accessed.
