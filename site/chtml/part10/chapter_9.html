9 Conformance Requirements

An implementation of PS3.10 shall:

  1. have a Conformance Statement based on a PS3.11 Application Profile in accordance with the framework defined in PS3.2;

  2. meet the requirements of the DICOM File Format as specified in Section 7;

  3. support the DICOM File Service as specified in Section 8, in one or more of the roles identified in Section 8.3;

  4. perform the Media Operations defined in Table 8.3-1 according to the role supported;

  5. support the DICOMDIR File with a content as specified in the Media Storage Directory SOP Class in PS3.4.
